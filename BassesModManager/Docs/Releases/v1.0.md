# Release 1 – v1.0

Initial release of Basse's Mod Manager.

## Summary

- Simplified mod manager for Star Wars Battlefront (competitive play), built on Frosty Toolsuite.
- Pre-selected set of approved mods; users choose which combination to run and launch the game from the app.
- In-app update check (version compared against `version.txt` on GitHub).
